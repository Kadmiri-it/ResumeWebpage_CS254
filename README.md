For this project, I created a simple and professional resume landing page using HTML and CSS, structured into clear sections including a header with my name, an About section, Skills, Experience, and Contact information. I maintained the project using GitHub with multiple commits to reflect each stage of development. To deploy the site, I connected my GitHub repository to Cloudflare Pages, configured the project with no build command since it is a static site, and ensured automatic deployment on each push. For the custom domain setup, I added my domain in the Cloudflare Pages dashboard and updated the DNS records by creating a CNAME pointing to the Cloudflare Pages URL. Cloudflare automatically handled the SSL configuration, enabling secure HTTPS access to my site. One challenge I faced was ensuring that all files were correctly linked and displayed after deployment, which I resolved by checking file paths and redeploying the project. Overall, this project helped me understand the full workflow from development to deployment.
